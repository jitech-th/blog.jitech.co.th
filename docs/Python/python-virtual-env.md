# Using python `virtualenv` as an isolate environment

## Instuctions

- #### Install virtualenv
```
$ sudo pip3 install virtualenv
```

- #### Create env
```
$ python3.6 -m virtualenv env-name
```

- #### Activate env
```
$ source selp-env/bin/activate
(selp-env) $
```
